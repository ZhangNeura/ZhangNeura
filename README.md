# Wei Zhang

**Building Neura — the intelligence layer for prediction markets.**

For every active market on Polymarket (and soon Kalshi, Limitless), 4 AI agents extract signals from public reality — news, SEC filings, on-chain transactions, social — and surface the gap between the venue's implied probability and the model's probability as a structured `edge_delta` that funds plug into their alpha models.

[**Live dApp →**](https://neuramarkets.lovable.app)  ·  [**neura source →**](https://github.com/ZhangNeura/neura)  ·  [**askneura.xyz →**](https://askneura.xyz)

</div>

---

### Currently focused on

- Multi-agent architectures with hard **point-in-time** guarantees (`tstzrange` + `EXCLUDE GIST`)
- Synthesizing **model probability** from news, filings, on-chain, social — without look-ahead bias
- **Token-gated** access models that don't leak the alpha
- A polyglot monorepo dev experience — TypeScript · Rust · Python · SQL
